Design a full ultra-premium architecture firm website called "ARCVAULT" — 
a luxury architectural studio. The design language should feel like a 
high-end editorial magazine crossed with brutalist concrete geometry. 
Dark, cinematic, with sharp gold/brass accents on near-black backgrounds.

TYPOGRAPHY:
- Display headings: Cormorant Garamond (thin/light weight, large tracking)
- Sub-headings: Neue Haas Grotesk or DM Sans (medium)
- Body: DM Sans Regular
- Accent labels: Bebas Neue or Letter Gothic (uppercase, wide tracking)

COLOR PALETTE:
- Background: #0A0A0A (near black)
- Primary text: #F5F0E8 (warm off-white)
- Gold accent: #C9A84C
- Muted tone: #1C1C1C (card/section bg)
- Subtle grid lines: #222222
- Highlight/glow: rgba(201, 168, 76, 0.12) soft gold glow

GLOBAL DESIGN RULES:
- All sections: 120px top/bottom padding (desktop), proper 80px on tablet
- Max content width: 1440px, centered
- Smooth scroll between sections
- Section transitions: subtle fade-in-up as each section enters viewport
- All hover states: ~300ms ease transitions
- Custom scrollbar: thin, gold thumb on dark track
- No border-radius on main cards — sharp corners only (architectural)
- Subtle noise/grain texture overlay across the entire site (4-5% opacity)
- All images: aspect-ratio locked, object-fit cover

RESPONSIVE NOTES:
- Desktop: Full layout as described
- Tablet (768px): Stack split sections, reduce heading sizes ~25%
- Mobile: Single column, hamburger nav, hero text scales to ~56px
